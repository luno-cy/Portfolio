Réalisations techniques

Des projets concrets en cybersécurité

Infrastructure Windows Sécurisée

SYS12

Conception et déploiement d'une infrastructure Windows Server virtualisée pour l'entreprise fictive NeoTech Solutions. Architecture redondée avec 2 contrôleurs de domaine, serveurs IIS, WSUS et serveur de fichiers. Organisation Active Directory structurée (OU, groupes, GPO) et politique de sécurité.

Windows Server Active Directory GPO IIS WSUS DNS/DHCP

Audit Offensif & Remédiation

OFF12

Audit de sécurité interne sur l'infrastructure NeoTech Solutions. Introduction contrôlée de 5 vulnérabilités (mots de passe faibles, configurations IIS, partages non sécurisés, ports exposés). Phases de scan, exploitation, analyse d'impact et correction complète avec documentation professionnelle.

Nmap Pentesting IIS NTFS Pare-feu

Infrastructure NovaBike

INT12

Déploiement complet d'une infrastructure sécurisée pour l'entreprise fictive NovaBike (10 salariés). Mise en place d'un contrôleur de domaine, serveur de fichiers avec permissions, pare-feu, politique de mots de passe et script de sauvegarde automatisé. Documentation professionnelle et schéma d'architecture.

Windows Server AD DS Pare-feu Sauvegarde PowerShell

Secure Messenger

CYR12

Conception d'une messagerie locale sécurisée avec authentification robuste et chiffrement des communications. Implémentation de hachage avec sel unique, chiffrement symétrique garantissant confidentialité et intégrité. Analyse du modèle de menace et test de résistance face aux attaques par dictionnaire.

Python Flask SQLite Cryptographie OWASP

Sécurisation de Base de Données

BDD11

Conception et implémentation d'une base de données sécurisée pour une plateforme de gestion de projets. Mise en œuvre du contrôle d'accès RBAC (rôles étudiant/tuteur), analyse et prévention des injections SQL via des requêtes paramétrées, et modélisation Merise (MCD/MLD).

PostgreSQL RBAC SQLi Prevention Merise Python

Réseau d'Entreprise Segmenté

NET11

Conception d'une architecture réseau sécurisée pour la PME InnovTech Solutions. Segmentation logique en zones (DMZ, LAN, VPN), configuration de pare-feu pfSense/OPNsense, mise en place de NAT/PAT, et audit de sécurité réseau avec analyse de flux (Wireshark, Nmap).

pfSense VPN VLAN NAT/PAT Wireshark

Hardening & Surveillance OS

SYS11

Création d'une VM Linux vulnérable pour l'étude des vecteurs d'attaque. Développement de scripts d'audit en Bash (Linux) et PowerShell (Windows), mise en place d'une surveillance active des journaux et alertes, et formulation de recommandations de durcissement (Hardening).

Linux Hardening Bash Scripting PowerShell Audit Système
